Python — объектно-ориентированный язык программирования, его основу составляют объекты и классы. Объект — это область памяти компьютера, которая описывается типом (он же класс) и значением. При этом от типа зависит область значений объекта, операции и методы, которые к нему можно применять. Python предоставляет богатый набор встроенных типов данных. Поэтому при решении стандартных задач питонист реже пишет собственные классы, чем, например, разработчик на Java.

Команда вывода в языке Python является 'print'

Ввод днных с клавиатуры является 'input'. Записывается это данным образом: тип данных (input())

Какие типы данных есть в Python?

  • Простые - числа и строки.
  • Коллекции - списки, кортежи.
  • Остальные - файлы, интераторы, сокеты, NaN.

Для начала нам нужно будет изучить 'простые типы данных'.

В них входит:

  • Тип 'int'
  • Тип 'float'
  • Тип 'string'
  • Тип 'bool'

Условия в Python
условия что из себя представляет
if В Python конструкция if используется для выполнения условных операторов. Она позволяет выполнять определенный блок кода только в том случае, если заданное условие истинно (то есть имеет значение True). Если условие ложно (False), код внутри блока if будет пропущен.
else В Python конструкция else используется в условных операторах для выполнения блока кода, если все предыдущие условия (в if и elif) были ложными. Это позволяет задать альтернативное действие, когда ни одно из условий не выполнено.
elif elif в Python — это сокращение от "else if". Он используется в условных операторах для проверки дополнительных условий, если предыдущее условие (в if) оказалось ложным.

Синтаксис:

if условие1:

# блок кода, выполняемый, если условие1 истинно

elif условие 2:

# блок кода, выполняемый, если условие2 истинно

else :

# блок кода, выполняемый, если ни одно из условий не истинно

После идут циклы не менее важная деталь в Python. Циклы бывают:

  • Цикол "for"
  • Цикол "while"
  • И команда "break"
Циклы
цикл и команда break что из себя представляет
for Цикл for в Python используется для итерации по элементам последовательностей, таких как списки, кортежи,строки или другие итерируемые объекты. Он позволяет выполнять блок кода несколько раз для каждого элемента в последовательности.
while Цикл while в Python используется для повторения блока кода до тех пор, пока заданное условие остается истинным. Это позволяет выполнять операции, пока определенное условие выполняется.
break Команда break в Python используется для немедленного выхода из цикла. Это может быть полезно, когда нужно завершить выполнение цикла до его естественного завершения, например, при выполнении определенного условия.

Циклы являются мощным инструментом в Python, позволяющим эффективно обрабатывать данные и выполнять повторяющиеся задачи. Понимание их работы поможет вам писать более гибкий и эффективный код.
