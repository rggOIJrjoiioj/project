Java — это строго типизированный объектно-ориентированный язык программирования общего назначения, разработанный компанией Sun Microsystems.

Команда вывода в языке Java является System.out.print() или println()

Ввод днных с клавиатуры является String str = console.nextLine(); или int number = console.nextInt(); или double number = console.nextDouble();

После идут циклы, не менее важная деталь в Java. Циклы бывают:

  • Цикл "for"
  • Цикл "while"
  • Цикл "do-while"
  • Улучшенный цикл for (foreach)
  • А так же команды "break" и " continue"

Каждая из этих концепций является важной частью языка Java и помогает в создании качественного, поддерживаемого и эффективного кода. Понимание этих основ — ключ к успешной разработке программного обеспечения. Если у вас есть вопросы или хотите обсудить какую-либо тему подробнее, не стесняйтесь спрашивать!

Циклы
цикл и команды что из себя представляет
for Цикл for используется, когда количество итераций известно заранее. Он состоит из трех частей: инициализации, условия и инкремента.
while Цикл while используется, когда количество итераций неизвестно и зависит от условия. Он выполняется до тех пор, пока условие истинно.
do-while Цикл do-while похож на while, но гарантирует, что блок кода выполнится хотя бы один раз, так как условие проверяется после выполнения блока кода.
for (foreach) Улучшенный цикл for позволяет легко перебрать элементы коллекции или массива.
break прерывает выполнение цикла.
continue пропускает текущую итерацию и переходит к следующей.

Задания для закрепления

Задание 1: Определение типа данных. Условие: Напишите программу, которая запрашивает у пользователя ввод значения и определяет его тип (целое число, дробное число или строка)..

import java.util.Scanner;
public class DataTypeChecker {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
System.out.print("Введите значение: ");
if (scanner.hasNextInt()) {
System.out.println("Это целое число.");
} else if (scanner.hasNextDouble()) {
System.out.println("Это дробное число.");
} else {
System.out.println("Это строка.");
}
scanner.close();
}
}

Задание 2: Таблица умножения Условие: Напишите программу, которая выводит таблицу умножения от 1 до 10.

public class MultiplicationTable {
public static void main(String[] args) {
System.out.println("Таблица умножения от 1 до 10:");
for (int i = 1; i <= 10; i++) {
for (int j = 1; j <= 10; j++) {
System.out.printf("%d x %d = %d\t", i, j, i * j);
}
System.out.println();
}
}
}

Задание 3: Сумма чисел Условие: Напишите программу, которая запрашивает у пользователя целое число N и вычисляет сумму всех целых чисел от 1 до N

import java.util.Scanner;
public class SumOfNumbers {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
System.out.print("Введите целое число N: ");
int N = scanner.nextInt();
int sum = 0;
for (int i = 1; i <= N; i++) {
sum += i;
}
System.out.println("Сумма чисел от 1 до " + N + " = " + sum);
scanner.close();
}
}

Задание 4: Фибоначчи Условие: Напишите программу, которая выводит первые N чисел последовательности Фибоначчи, где N вводится пользователем.

import java.util.Scanner;
public class FibonacciSequence {
public static void main(String[] args) {
Scanner scanner = new Scanner(System.in);
System.out.print("Введите количество чисел Фибоначчи для вывода: ");
int N = scanner.nextInt();
int a = 0, b = 1;
System.out.println("Последовательность Фибоначчи:");
for (int i = 1; i <= N; i++) {
System.out.print(a + " ");
int next = a + b;
a = b;
b = next;
}
scanner.close();
}
}
